# solder-cupcake
I made a cupcake design for the PCB that has lights for sprinkles and a button that starts the motor to vibrate.

Materials used:
- 4 LEDs
- 5 resistors
- 1 Battery Holder
- 1 Push Button
- 1 DC Motor
